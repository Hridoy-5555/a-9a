🩺 DocAppoint — Doctor Appointment Manager
DocAppoint হলো একটি আধুনিক এবং সম্পূর্ণ রেসপনসিভ ডক্টর অ্যাপয়েন্টমেন্ট বুকিং প্ল্যাটফর্ম (Single Page Application)। এই সিস্টেমের মাধ্যমে রোগীরা সহজেই অভিজ্ঞ ডাক্তারদের প্রোফাইল ব্রাউজ করতে পারবেন, বিস্তারিত তথ্য দেখতে পারবেন এবং সুরক্ষিতভাবে অ্যাপয়েন্টমেন্ট বুক করতে পারবেন। এছাড়া ব্যবহারকারীরা তাদের ড্যাশবোর্ড থেকে বুকিং ও প্রোফাইল সরাসরি পরিচালনা করতে পারবেন।

🔗 গুরুত্বপূর্ণ লিংকসমূহ
লাইভ সাইট ইউআরএল (Live Site): এখানে আপনার লাইভ সাইটের লিংক দিন

সার্ভার-সাইড গিটহাব রিপোজিটরি: এখানে সার্ভার সাইড গিটহাব লিংক দিন

🚀 প্রধান বৈশিষ্ট্যসমূহ (Key Features)
JWT এবং সেশন ভিত্তিক সুরক্ষিত অথেন্টিকেশন: Better Auth এবং JWT টোকেন ব্যবহার করে প্রজেক্টের সিকিউরিটি নিশ্চিত করা হয়েছে। এর ফলে প্রাইভেট রাউটে থাকা অবস্থায় পেজ রিলোড করলেও ইউজার লগইন পেজে রিডাইরেক্ট হবে না।

ডাইনামিক বুকিং এবং ইনস্ট্যান্ট নোটিফিকেশন: কোনো প্রকার ডিফল্ট ব্রাউজার অ্যালার্ট ছাড়াই, সম্পূর্ণ কাস্টমাইজড টোস্ট (Toast) মেসেজের মাধ্যমে অ্যাপয়েন্টমেন্ট বুকিং, আপডেট এবং ডিলিট করার রিয়েল-টাইম ফিডব্যাক দেওয়া হয়।

অ্যাডভান্সড সার্চ এবং সর্টিং (Sorting): 'All Appointments' পেজে ডাক্তারদের নাম দিয়ে ডাইনামিকালি সার্চ করার সুবিধা রয়েছে। পাশাপাশি রোগীদের সুবিধার্থে অ্যাপয়েন্টমেন্টগুলো সর্ট করার অপশন যুক্ত করা হয়েছে।

পূর্ণাঙ্গ ড্যাশবোর্ড (CRUD Operations): লগইন করা ইউজাররা তাদের নিজস্ব ড্যাশবোর্ড থেকে বুক করা অ্যাপয়েন্টমেন্টের তারিখ ও সময় পরিবর্তন (Update) এবং ক্যানসেল (Delete) করতে পারবেন। ডাটা সুরক্ষার জন্য এডিটের সময় ডাক্তারের তথ্য এবং ইউজারের ইমেইলটি Read-only মোডে থাকে।

ইউনিক ইউজার ইন্টারফেস ও ডার্ক মোড: কোনো প্রকার 'Lorem Ipsum' টেক্সট ছাড়া বাস্তবসম্মত কন্টেন্ট ব্যবহার করা হয়েছে। হোম পেজে ইন্টারেক্টিভ Swiper.js স্লাইডার, আধুনিক গ্রিড লেআউট এবং একটি আকর্ষণীয় লাইট/ডার্ক থিম টগল সিস্টেম রয়েছে।

🛠️ ব্যবহৃত টেকনোলজি (Tech Stack)
ফ্রন্টএন্ড: React.js, React Router DOM (v6), Tailwind CSS

অথেন্টিকেশন: Better Auth (JWT-based session management), Google/GitHub Social Login

প্যাকেজ ও লাইব্রেরি: Swiper.js (ব্যানার স্লাইডার), React Hot Toast / Sonner (ইনস্ট্যান্ট নোটিফিকেশন), React Icons (নতুন X লোগো সহ)

ব্যাকএন্ড ও ডাটাবেজ: Node.js, Express.js, MongoDB
